Istruzioni rapide per usare novita.html
-------------------------------------

Files inclusi:
- index.html  -> home con pulsante "Novità ed Eventi"
- novita.html -> pagina dove inserire le novità

Come aggiornare le novità:
1) Apri novita.html nel tuo editor GitHub (o sul PC).
2) Trova un blocco <article> (esempio già presente) e copialo.
3) Incolla il blocco sotto l'ultimo articolo e modifica:
   - <h2> Titolo
   - la riga con 'meta' (data o stato)
   - il paragrafo di descrizione
4) Salva/Commit su GitHub nella root del branch pubblicato (main).
5) Attendi 1-2 minuti e verifica su https://riccardomariopiane.it/novita.html

Nota sui link:
- Il pulsante "Iscriviti via email" apre la mail precompilata a rm.piane@yahoo.com.
- Il pulsante "Dove ricevo" punta a contatti.html (assicurati che il file esista nella root).
